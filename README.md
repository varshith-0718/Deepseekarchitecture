# DeepSeek Architecture Exploration

Exploration and implementation of concepts from the DeepSeek family of language models, focused on understanding its efficient attention mechanisms and Mixture-of-Experts (MoE) routing design.

## Goal

Studying and reproducing key architectural ideas from DeepSeek's published models to build deeper intuition for modern, compute-efficient LLM design choices.
